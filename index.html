  • Completed ZTM sections 9 and 10 - CSS Grid and CSS Layout; Career of a Developer. The CSS Grid and Layout section was trickier than I expected, once I got to the exercise where we were asked to create a webpage layout (lesson 97). I found a working solution for the header, cover and footer but gave up on figuring out the portfolio grid and went to the solution video in the end, as I was just getting lost in googling things and getting frustrated. I was happy to see that I'd got some things right, but I feel that I need to study the correct structure and go from there. I'd gotten in a mess by turning the whole page into a grid and then trying to create another grid inside it, not realising that in this case only the portfolio section needed to be a grid. I also wasn't sure if I should be using flexbox for that section and spent hours fiddling about trying to do that at first. It felt like my whole brain was made up of disconnected divs at one point!! I'd also done some other quirky things and created a weird horizontal overflow. I like this course, but the emphasis on whisking quickly through the HTML and CSS to get onto the JS (as this is the bit employers will be interested in) is going against my usual approach of learning things thoroughly and I'm struggling with some imposter syndrome about finding CSS layouts quite difficult. Very glad I already know a tiny bit about Git/GitHub and have programmed a little bit in C, so the upcoming sections don't feel completely unfamiliar!
